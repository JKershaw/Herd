# Herd

A pixel-art sheep herding game built with React and Canvas.

Command your sheepdog with whistle commands to guide the flock into the pen.

## Controls

- **Q/1** — Come-bye (circle left)
- **W/2** — Away to me (circle right)
- **E/3** — Walk up (approach)
- **R** — Restart
- **Space** — Start game

## Play

Open `index.html` in a browser, or visit the GitHub Pages site.
